Criar Macro

Como criar macros:

  • Uma linha: 1d20+5 → Rola 1d20 + 5
  • Múltiplas linhas: Uma por vez, cada linha é uma rolagem separada
  • Atributos: Use o nome do stat, ex: 1d20+for ou 1d20+Força
  • Comparação: 1d100<=Int → Mostra SUCESSO ou FALHA
  • Pool: 5d6>4 → Conta dados ≥ 4
  • Explosão: 4d6! → Rola extras em 6
  • Vantagem: 2d20kh1 → Mantém maior de 2d20
  • Repetição: 3d6x2 → Rola 3d6 duas vezes

Para testar, clique em "Testar Fórmula" antes de salvar.
